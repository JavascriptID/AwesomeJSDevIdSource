Awesome JS Dev Indonesia

version 1.0.0 @ Gulajava {{datePrivacyPolicy}}

Simple web application that contains a list of JavaScript developers and programmers from Indonesia that are AWESOME . These data are sourced from the Awesome JS Dev ID Github repository which can be accessed here https://github.com/mazipan/awesome-js-dev-id. If you want to add data or make data corrections, please make corrections with the Pull Request to the repository. This Awesome JS Dev Indonesia application source code is on the following Github https://github.com/javascript-indonesias/AwesomeJSDevIdSource . You can see it for use in learning and examples of web implementation with PWA's Progressive Web Apps.
